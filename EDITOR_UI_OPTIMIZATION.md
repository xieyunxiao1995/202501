# 编辑器页面 UI 优化

## 主要改进

### 1. 标题优化
**之前：** "新记忆"
**现在：** "记录此刻"

- 更加贴近用户的使用场景
- 添加了带渐变背景的图标
- 优化了日期和字数统计的显示样式
- 使用图标增强视觉层次

### 2. 保存按钮优化
**之前：** "发牌"（不够直观）
**现在：** "保存"（清晰明确）

- 使用心情颜色的渐变背景
- 添加了确认图标
- 有内容时显示阴影效果
- 禁用状态更加明显

### 3. 心情选择区域优化
- 添加了"此刻的心情"标题和图标
- 使用渐变背景和边框
- 边框颜色跟随选中的心情变化
- 整体视觉更加精致

### 4. 文本输入区域优化
- 添加了白色半透明背景
- 更详细的提示文字：
  ```
  写下此刻的想法...
  
  可以是今天发生的事，内心的感受，或是任何想要记录的瞬间
  ```
- 优化了字体间距和行高
- 添加了圆角边框

### 5. 隐私开关优化
**之前：** 简单的滑动开关
**现在：** 带标签的按钮样式

- 显示"私密"或"公开"文字
- 使用图标和文字组合
- 公开状态使用欢快的颜色
- 更加直观易懂

### 6. 图片按钮优化
- 改用轮廓图标样式
- 添加了背景色
- 添加了工具提示
- 视觉上更加统一

## 用户体验提升

1. **更清晰的操作指引**
   - "保存"比"发牌"更直观
   - 提示文字更加详细友好

2. **更好的视觉反馈**
   - 按钮状态变化更明显
   - 心情颜色贯穿整个界面
   - 渐变和阴影增强层次感

3. **更统一的设计语言**
   - 所有元素都使用圆角设计
   - 颜色系统更加协调
   - 图标和文字搭配使用

4. **更友好的交互**
   - 隐私开关一目了然
   - 保存按钮有明确的视觉提示
   - 输入区域更加舒适

## 技术实现

- 使用 `LinearGradient` 创建渐变效果
- 使用 `BoxShadow` 增强立体感
- 动态颜色系统跟随心情变化
- 保持了原有的动画效果
