# 一策定江山

> 运筹帷幄，逐鹿中原

**一策定江山** 是一款以三国为题材的回合制策略卡牌 RPG，基于 Flutter 构建，支持 iOS / Android 双平台。玩家扮演乱世主公，招募名将、排兵布阵、施展计谋，亲历官渡、赤壁、夷陵等经典战役，最终一统天下。

---

## 游戏特色

| 特色 | 说明 |
|------|------|
| 🎴 **名将收集** | 收录赵云、关羽、周瑜等数十位三国名将，SSR/UR 多稀有度 |
| 🧠 **谋略系统** | 区别于纯数值堆砌，「计谋」是核心策略——火攻、水淹、离间、伏兵 |
| ⚔️ **兵种相克** | 盾兵 > 弓兵 > 枪兵 > 骑兵 > 盾兵，搭配阵法布局还原古代战争 |
| 🏰 **势力争霸** | 跨服国战，魏蜀吴群雄逐鹿，公会联盟合纵连横 |
| 🔗 **武将羁绊** | 还原历史关系——桃园结义、五虎上将、东吴四英将 |
| 📖 **双线叙事** | 演义线与正史线并行，同一角色不同时间线展现不同面貌 |

---

## 核心玩法

```
招募武将 → 组建阵容 → 排布阵法 → 施展计谋 → 挑战关卡
    │                                      │
    └──────── 养成：升级 / 突破 / 觉醒 ←────┘
```

- **关卡征战**：15 个章节，从黄巾之乱到三家归晋，每章 3 个战役节点
- **主城建设**：酒馆、学堂、铁匠铺、马厩等功能建筑
- **背包系统**：道具 / 装备 / 材料 / 宝物四大分类
- **招募系统**：选择至多 4 名武将出战

## 技术栈

```
Flutter 3.19+    │ 跨平台 UI 框架
Dart 3.3+        │ 编程语言
Riverpod 2.5+    │ 状态管理
GoRouter          │ 路由导航
Dio + WebSocket   │ 网络通信
Hive CE           │ 本地存储
Flame 1.16+       │ 战斗场景引擎
Rive / Spine      │ 武将骨骼动画
```

## 快速开始

```bash
# 安装依赖
flutter pub get

# 代码生成（freezed / json_serializable / riverpod / flutter_gen）
dart run build_runner build --delete-conflicting-outputs

# 运行
flutter run

# 运行测试
flutter test
```

## 工程结构

```
lib/
├── main.dart              # 应用入口
├── app.dart               # MaterialApp 配置
├── core/                  # 基础设施层（路由、主题、存储、网络）
├── data/                  # 数据层（模型、数据源、映射器）
├── domain/                # 领域层（实体、仓储接口、用例）
├── game/                  # 战斗引擎（Flame 场景、ECS 组件）
├── pages/                 # 页面层（关卡、背包、招募、主城各建筑）
├── presentation/          # 展示层（Widget 组件、Notifier 状态）
├── shared/                # 共享组件
└── generated/             # 代码生成产物
```

## 页面总览

| 模块 | 页面 | 路径 |
|------|------|------|
| 启动 | 闪屏 | `/splash` |
| 认证 | 登录 | `/login` |
| 主城 | 主界面 | `/city` |
| 关卡 | 章节征战 / 关卡详情 | `/chapter/:chapterId` / `/stage/:chapterId/:stageId` |
| 招募 | 选择武将 | `/recruit` |
| 背包 | 背包 | `/package` |
| 建设 | 酒馆 / 学堂 / 铁匠铺 / 马厩 / 演武场 | `/city/tavern` 等 |

## 配置数据

游戏数值通过 `assets/data/` 下的 JSON 文件驱动，支持热更新：

| 文件 | 内容 |
|------|------|
| `generals.json` | 武将属性、技能、稀有度 |
| `skills.json` | 技能效果、伤害系数 |
| `tactics.json` | 计谋配置 |
| `formations.json` | 阵法加成 |
| `stages.json` | 关卡、敌军、掉落 |
| `equipment.json` | 武器、铠甲、宝物 |
