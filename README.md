# 星悦

> 每一件衣服，都藏着一段心情。

## 简介

星悦是一款陪伴你发现自我风格的穿搭日记应用。

我们相信，穿搭从来不只是关于衣服——它是你与世界对话的方式，是内心情绪的外在表达。那些你精心挑选的衬衫、随手披上的围巾、反复穿上的旧球鞋，都在无声地诉说着你是谁。

星悦帮你记录这些日常片段：今天的穿搭、此刻的心情、偶然闪现的灵感。时间久了，你会看见自己的风格轨迹——原来阴天偏爱柔软，晴天倾向利落；原来某些颜色总与某些情绪相伴。

这里没有标准答案，只有真实的你。

## 核心功能

- **穿搭日记**：用照片记录每日穿搭，建立专属风格档案
- **心情记录**：关联穿搭与情绪，发现风格与心境的微妙联系
- **AI 穿搭助手**：智能对话，获取个性化的穿搭建议与灵感
- **社区分享**：与同好分享穿搭故事，发现更多生活可能
- **风格探索**：通过数据分析，看见自己的穿搭偏好与变化轨迹
