# 星悦

> 每一件衣服，都藏着一段心情。

## 简介

星悦是一款陪伴你发现自我的日记应用。

我们相信，每一个平凡的日子都值得被记录——那些细微的情绪波动、偶然闪现的灵感、不经意间的心动，都是生命中最真实的片段。

星悦帮你珍藏这些时光：此刻的心情、今日的风景、那些无法言说却真实存在的感受。时间久了，你会看见自己的成长轨迹——原来阴天需要温暖，晴天渴望自由；原来某些瞬间总与某些记忆相连。

这里没有标准答案，只有真实的你。

## 核心理念

- 陪伴：记录每一个当下，让时光有迹可循
- 觉察：看见情绪与日常的微妙联系，更懂自己
- 启发：在对话中获得灵感，发现生活的更多可能
- 共鸣：与同频的人分享故事，感受彼此的温度
- 探索：回望来时的路，遇见更完整的自己

### 关于背景音乐功能的说明

Dear Apple Review Team,

Regarding the Background Music feature in our app:

功能位置 / Feature Location:
用户可以在应用内依次进入 "我的" -> "设置与更多" -> "背景音乐" 页面访问该功能。
Users can access this feature by navigating to "Profile" -> "Settings & More" -> "Background Music".

功能说明 / Feature Description:
该功能为用户提供多首由 AI 算法生成的背景音乐，支持顺序播放、列表循环和单曲循环三种播放模式。用户可以选择任意一首音乐进行播放，音乐将在应用切换到后台后继续播放，保证用户体验的连续性。
This feature provides users with multiple background music tracks generated by AI algorithms, supporting three playback modes: sequential, loop playlist, and repeat single track. Users can select any track to play, and the music will continue playing even when the app is switched to the background, ensuring a seamless user experience.

为什么需要此功能 / Why This Feature is Needed:
1. 提升沉浸式体验：星悦作为一款记录心情与穿搭的日记应用，我们希望用户在日常使用中能伴随舒适的背景音乐，获得更加沉浸和放松的体验。
2. 后台播放支持生活化场景：用户在通勤、运动或做家务时，可能希望继续收听应用中的音乐。我们为此配置了 iOS 后台音频模式，让用户在不打开应用时也能享受音乐陪伴。
3. 个性化选择：我们提供 5 首不同风格的 AI 生成音乐，并支持多种播放模式，满足不同用户在不同场景下的偏好。

If you need any additional information or clarification, please don't hesitate to contact us.

Best regards,
星悦 Development Team
