# 心屿 MindFlow - 快速启动指南

## 🚀 5分钟快速开始

### 1. 环境准备
确保已安装：
- Flutter SDK 3.9.2+
- Dart SDK 3.9.2+
- iOS模拟器或Android模拟器

### 2. 安装依赖
```bash
flutter pub get
```

### 3. 配置API Key
编辑 `lib/services/deepseek_service.dart`：
```dart
static const String apiKey = 'your-deepseek-api-key';
```

### 4. 运行应用
```bash
flutter run
```

## 📱 首次使用

### 第一步：创建日记
1. 点击底部中央的 **+** 按钮
2. 选择今天的心情（开心/平静/焦虑/忧郁）
3. 输入日记内容
4. 可选：添加图片
5. 点击"发牌"保存

### 第二步：查看日记
1. 在主页左右滑动浏览日记
2. 点击刷新图标翻转卡片，查看AI洞察
3. 点击堆叠卡片查看该组所有日记

### 第三步：与AI对话
1. 点击底部"伴侣"图标
2. 选择快捷回复或输入消息
3. 与AI分享你的想法和感受

### 第四步：浏览广场
1. 点击底部"广场"图标
2. 浏览其他人的心情分享
3. 点击卡片为喜欢的内容点赞

### 第五步：个性化设置
1. 点击底部"设置"图标
2. 配置通知提醒时间
3. 选择喜欢的主题
4. 定期导出数据备份

## 🔧 常见问题

### Q: 应用启动失败？
A: 检查Flutter版本是否为3.9.2+，运行 `flutter doctor` 检查环境

### Q: AI功能不工作？
A: 确认已配置正确的DeepSeek API Key

### Q: 如何删除单篇日记？
A: 当前版本暂不支持，可使用"清空所有数据"重置

### Q: 数据存储在哪里？
A: 使用SharedPreferences存储在本地设备

### Q: 如何备份数据？
A: 使用设置中的"数据导出"功能
