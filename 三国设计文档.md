**适用平台：iOS / Android**

## 一、游戏概述

### 1.1 游戏名称
**《三国谋定天下》**（暂定名）
副标题：*运筹帷幄，逐鹿中原*

### 1.2 游戏类型
回合制策略卡牌 RPG（Turn-Based Strategy Card RPG）

### 1.3 题材定位
以《三国演义》为核心 IP，融合正史与演义，构建一个"乱世出英雄，谋略定乾坤"的三国世界。玩家扮演一位穿越乱世的"主公"，招募名将、组建势力、参与重大战役，最终一统天下。

### 1.4 目标用户
- **核心用户**：25-45 岁男性，三国题材爱好者、卡牌策略游戏玩家、有付费能力的中重度用户
- **泛用户**：18-35 岁泛用户，喜欢历史题材、武将收藏、策略对战
- **历史向用户**：对三国历史有浓厚兴趣，喜欢"考据"和"还原"的玩家

### 1.5 核心卖点
1. **经典三国 IP**：耳熟能详的武将、战役、典故
2. **谋略系统**：区别于纯数值堆砌，"计谋"是核心策略
3. **兵种相克 + 阵法布局**：还原古代战争的策略性
4. **势力争霸**：跨服国战，重现"群雄逐鹿"
5. **武将羁绊**：还原历史关系（桃园三结义、五虎上将等）
6. **轻度操作 + 重度策略**：碎片化游玩，深度养成

### 1.6 商业模式
F2P（免费下载）+ IAP（内购）+ 月卡/赛季通行证

## 二、世界观与剧情设定

### 2.1 世界观背景

> 东汉末年，黄巾乱起，天下分崩。董卓乱政，群雄并起，逐鹿中原。
>
> 玩家扮演一位身负"传国玉玺秘密"的乱世豪杰，于乱世中崛起，招贤纳士、攻城略地、合纵连横。在历史的洪流中，你将与曹操、刘备、孙权等枭雄并肩或为敌，亲历官渡、赤壁、夷陵等经典战役，最终成就属于自己的霸业。
>
> 游戏融合"正史"与"演义"双线叙事：白天演义，夜晚正史。同一个角色，在不同时间线展现不同面貌。

### 2.3 双线叙事设计
- **演义线**：还原《三国演义》经典桥段，富有戏剧性（如温酒斩华雄、草船借箭）

- 玩家可解锁双重剧情，获得不同奖励

## 三、核心玩法系统

### 3.1 战斗系统

#### 3.1.1 战斗形式
- **6v6 回合制对战**
- **横版站位**：前军（2位）、中军（2位）、后军（2位）
- **回合制**：根据武将"速度"属性决定行动顺序
- **战斗时长**：单局 1-3 分钟（PVE），2-5 分钟（PVP）

#### 3.1.2 兵种系统
六大兵种相克，类似石头剪刀布的升级版：

```
步兵 → 弓兵 → 骑兵 → 步兵（基础三角）
枪兵 → 骑兵 → 盾兵 → 枪兵（克制关系）
谋士 → 全体（无视兵种相克，但血量低）
```

| 兵种 | 特点 | 克制 | 被克 |
|------|------|------|------|
| 步兵 | 平衡型 | 弓兵 | 骑兵 |
| 骑兵 | 高机动高攻击 | 步兵、弓兵 | 枪兵 |
| 弓兵 | 远程输出 | 骑兵 | 步兵 |
| 枪兵 | 反骑专家 | 骑兵 | 盾兵 |
| 盾兵 | 高防御坦克 | 枪兵 | 弓兵 |
| 谋士 | 法术输出，无视相克 | 全体 | 自身脆弱 |

**克制效果**：造成伤害 +30%，受到伤害 -20%

#### 3.1.3 站位机制
| 站位 | 特性 | 适合武将 |
|------|------|----------|
| 前军 | 受到物理伤害+15%，承担敌方近战嘲讽 | 坦克、近战武将 |
| 中军 | 平衡位置，无加成 | 战士、刺客 |
| 后军 | 受到物理伤害-20%，但被穿透技能优先攻击 | 谋士、弓兵、辅助 |

#### 3.1.4 技能体系
每个武将拥有 **4 类技能**：

1. **普通攻击**：每回合自动释放，无消耗
2. **战法（主动）**：消耗 2 怒气，主动技能
3. **大招（必杀技）**：消耗 4 怒气，大范围/高伤害
4. **被动技能**：常驻效果或触发型（如"虎啸"：受到伤害低于30%时攻击+50%）

**怒气获取规则**：
- 每回合 +1
- 受到伤害 +1（上限 2/回合）
- 击杀敌人 +2

#### 3.1.5 计谋系统（特色机制）
区别于普通技能，"计谋"是三国题材独有的玩法：

| 计谋类型 | 效果 | 代表武将 |
|----------|------|----------|
| **火攻** | 群体灼烧伤害，持续 3 回合 | 周瑜、陆逊 |
| **水攻** | 群体减速 + 持续掉血 | 关羽（水淹七军） |
| **空城计** | 让敌方无法攻击我方后军 1 回合 | 诸葛亮 |
| **离间计** | 让敌方武将互相攻击 1 次 | 贾诩、陈宫 |
| **苦肉计** | 我方武将受伤但获得永久攻击加成 | 黄盖 |
| **借刀杀人** | 反弹伤害给随机敌方 | 司马懿 |
| **诱敌深入** | 引诱敌方主动攻击，触发我方反击 | 陆逊 |
| **声东击西** | 攻击前排，实际伤害后排 | 张辽 |

**计谋触发条件**：
- 怒气充满
- 战场上有特定条件（如有友军阵亡）
- 与特定武将同上场（羁绊触发）

#### 3.1.6 阵法系统
玩家可选择阵法，影响全队属性和站位效果：

| 阵法 | 效果 | 解锁条件 |
|------|------|----------|
| **一字长蛇阵** | 全体速度+15% | 主公等级 20 |
| **二龙出水阵** | 前军攻击+20% | 主公等级 30 |
| **三才阵** | 全体物理防御+15% | 主公等级 40 |
| **四象阵** | 中军血量+25% | 主公等级 50 |
| **五行阵** | 全体五行属性增强 | 主公等级 60 |
| **六合阵** | 后军法术防御+25% | 主公等级 70 |
| **七星阵** | 谋士技能伤害+30% | 主公等级 80 |
| **八卦阵** | 受到伤害有30%概率反弹 | 主公等级 90 |
| **九宫阵** | 全属性+10% | 主公等级 100 |

#### 3.1.7 战斗特殊机制
- **羁绊系统**：特定武将同时上场触发羁绊技能
  - 桃园结义（刘备+关羽+张飞）：全体攻防+20%
  - 五虎上将（关张赵马黄）：触发"五虎咆哮"群攻技
  - 一吕二赵三典韦（吕布+赵云+典韦）：单挑必胜
  - 卧龙凤雏（诸葛亮+庞统）：计谋伤害+50%
- **军师系统**：每场战斗指定一位"军师"，触发军师技能
- **武将单挑**：触发概率事件，两名武将单挑，胜者获得增益

### 3.2 武将（卡牌）系统

#### 3.2.1 武将稀有度
| 稀有度 | 颜色 | 获取难度 | 武将举例 |
|--------|------|----------|----------|
| N | 白色 | 极易 | 普通士兵、无名校尉 |
| R | 绿色 | 易 | 二线武将（如纪灵、华雄） |
| SR | 蓝色 | 中 | 一线武将（如夏侯惇、太史慈） |
| SSR | 紫色 | 难 | 名将（如张辽、甘宁、张郃） |
| UR | 橙色 | 极难 | 顶级名将（如关羽、张飞、周瑜） |
| **传说** | 红色金边 | 限定 | 五虎上将、卧龙凤雏、奸雄曹操 |

#### 3.2.2 武将阵营
划分为 6 大阵营，便于羁绊和剧情设计：

- **魏**：曹操、夏侯惇、张辽、典韦、许褚、司马懿、郭嘉等
- **蜀**：刘备、关羽、张飞、赵云、马超、黄忠、诸葛亮、庞统等
- **吴**：孙权、周瑜、陆逊、吕蒙、甘宁、太史慈、大乔小乔等
- **群**：吕布、董卓、袁绍、貂蝉、张角、华佗、左慈等
- **晋**：司马师、司马昭、邓艾、钟会、羊祜（后期解锁）
- **女将**：貂蝉、孙尚香、大乔、小乔、蔡文姬、祝融夫人

#### 3.2.3 职业分类
| 职业 | 定位 | 代表武将 |
|------|------|----------|
| **猛将** | 高攻高血，前排输出 | 张飞、典韦、许褚 |
| **守将** | 高血高防，承伤 | 曹仁、夏侯惇、廖化 |
| **武圣** | 攻防均衡，全能型 | 关羽、赵云 |
| **刺客** | 高爆发，单体秒杀 | 马超、甘宁 |
| **军师** | 群体输出，控制 | 诸葛亮、周瑜、司马懿 |
| **辅助** | 治疗、增益 | 华佗、蔡文姬 |
| **奇谋** | 计谋特化，特殊效果 | 贾诩、郭嘉、庞统 |

#### 3.2.4 武将养成
- **升级**：消耗经验书（上限随主公等级开放）
- **进阶**：消耗同名武将碎片 + 进阶石（N→R→SR→SSR→UR）
- **觉醒**：满级满阶后觉醒，解锁第 5 技能 + 立绘升级（"觉醒立绘"展示武将巅峰状态）
- **星级**：1-7 星，影响基础属性
- **武器装备**：每个武将 6 件装备（武器/头盔/铠甲/战靴/兵书/虎符）
- **专属兵器**：每个名将有专属兵器（如关羽的青龙偃月刀、张飞的丈八蛇矛），可强化升级
- **战马系统**：UR 及以上武将可装备战马（赤兔、的卢、绝影等）

#### 3.2.5 武将传记
每位武将都有详细的"武将传记"，包含：
- 历史生平（基于《三国志》）
- 演义典故（基于《三国演义》）
- 解锁条件：武将达到特定等级/觉醒后解锁
- 提供"考据"体验，增加历史代入感

### 3.3 PVE 玩法

#### 3.3.1 主线战役
- 章节制推进，按历史顺序展开
- 每章约 10-15 个关卡
- 三星评价制（满血通关 / 限时通关 / 阵容限制等）
- 通关奖励：武将碎片、装备、金币、剧情解锁

#### 3.3.2 经典战役副本
还原历史名战，特殊机制：

| 战役 | 特殊机制 |
|------|----------|
| **虎牢关之战** | 必须用 3 个武将"轮流单挑"吕布 |
| **官渡之战** | 火烧乌巢限时玩法 |
| **赤壁之战** | 火攻 + 东风 buff |
| **七擒孟获** | 必须 7 次抓获孟获才能通关 |
| **空城计** | 不能主动攻击，只能防御 |
| **草船借箭** | 限时收集箭矢 |

#### 3.3.3 精英副本
- 每日限次挑战
- 掉落特定材料（觉醒石、专属装备）

#### 3.3.4 爬塔玩法：群雄演武
- 100 层无尽塔
- 每 10 层一个名将 BOSS
- 排行榜奖励，月度重置

#### 3.3.5 资源副本
- **金币副本（金山）**：周一/周四开放
- **经验副本（讲武堂）**：周二/周五开放
- **装备副本（武器库）**：周三/周六开放
- **觉醒副本（名将冢）**：周日开放

#### 3.3.6 限时活动
- **群雄会武**：周常 BOSS 战，公会协作
- **节日活动**：春节、中秋、端午等推出限定武将
- **历史事件**：以"事件"形式展开（如"三英战吕布"、"七进七出"）

### 3.4 PVP 玩法

#### 3.4.1 校场切磋（异步 PVP）
- 玩家可设置防守阵容
- 攻击他人阵容获取排名
- 每日免费挑战 5 次，可购买次数
- 段位制：白丁→士卒→校尉→将军→大将→统帅→霸主

#### 3.4.2 群雄逐鹿（实时 PVP）
- 实时匹配同段位玩家
- BAN/PICK 机制（禁用 2 个武将，选择 6 个）
- 赛季制，每赛季 2 个月，赛季奖励丰厚

#### 3.4.3 国战玩法：天下争霸（特色玩法）
- 跨服大型 PVP 战场
- 玩家可加入 **魏蜀吴** 三国之一（自由选择）
- 占领城池、抢夺资源
- 每周固定时间开战（周五-周日）
- 国战奖励：国战币（兑换稀有武将）、城池税收、爵位（白身→男爵→侯爵→公爵→王爷）

#### 3.4.4 公会战
- 公会 vs 公会，跨服匹配
- 攻城战玩法，占领据点
- 周末开放

### 3.5 主城建设系统（特色玩法）

> "兵马未动，粮草先行。"

每位玩家拥有自己的"主城"，可建设发展：

#### 3.5.1 主要建筑
| 建筑 | 功能 |
|------|------|
| **主公府** | 主城等级，限制其他建筑等级 |
| **校场** | 训练士兵，提升武将基础属性 |
| **议事厅** | 接取任务，外交事务 |
| **武器坊** | 打造装备 |
| **马厩** | 培养战马 |
| **酒馆** | 招募武将（抽卡入口） |
| **粮仓** | 储存粮食（货币之一） |
| **铸币司** | 产出金币 |
| **学堂** | 培养军师，研究战法 |
| **观星台** | 占卜（每日免费抽奖） |
| **演武场** | PVP 入口 |

#### 3.5.2 资源采集
- **粮食**：粮仓产出，用于消耗体力
- **金币**：铸币司产出
- **木材/铁矿**：用于建造和装备打造
- **威望**：通过完成任务获取，提升爵位

### 3.6 武将招募系统（抽卡）

#### 3.6.1 招募池设计
| 卡池 | 内容 | 抽取方式 |
|------|------|----------|
| **新手招贤** | 必出 SSR 武将 | 仙玉/招募令 |
| **常驻招贤** | 全武将池 | 招募令 |
| **限定招贤** | 限定 UR 武将 | 限定招募令 |
| **阵营招贤** | 特定阵营武将（魏/蜀/吴/群） | 阵营令 |
| **军师招贤** | 军师类武将 UP | 军师令 |
| **传说招贤** | 传说武将 UP（限时） | 钻石 + 传说招募令 |

#### 3.6.2 保底机制
- **十连招募**：必出 SR 及以上
- **大保底**：90 次内必出 UR
- **传说保底**：180 次内必出传说武将
- **限定 UP**：50% 概率为当期 UP

## 四、经济与付费系统

### 4.1 主要货币
| 货币 | 获取方式 | 用途 |
|------|----------|------|
| 金币 | 战斗、副本 | 升级、强化装备 |
| 元宝（付费） | 充值、活动 | 招贤、购买体力 |
| 仙玉（免费） | 任务、活动 | 商店购买、招贤 |
| 体力（粮草） | 时间恢复、道具 | 进入副本 |
| 荣誉 | PVP 获得 | 校场商店 |
| 国战币 | 国战获得 | 国战商店（兑换稀有武将） |
| 公会贡献 | 公会任务 | 公会商店 |

## 十、风险评估与对策

| 风险 | 影响 | 对策 |
